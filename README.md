dOgeS
============================================

A browser-based OS project for CMPT 424 (Operating Systems), Fall 2015.


Setup
================

1. Fork or clone this repo.
2. Open index.html in the root directory of this project on your favorite browser (hopefully not IE...)
3. Success!


FAQs
